scMMDB Documentation
------------------------

**scMMDB: a comprehensive resource and knowledgebase for multi-omics data at the single-cell resolution**

.. .. image:: _static/logo.png
..     :align: center
..     :alt: scMMDB

.. figure:: _static/logo.png
   :align: center
   :alt: logo

   Single-cell Multimodal Omics DataBase

Single-Cell Multimodal Omics DataBase (scMMDB) is a comprehensive repository for general paired single-cell multimodal-omics data, 
encompassing both epigenomics, transcriptomics and proteomics data, and provides dataset/gene/protein search, visualization and download with unified formats.
